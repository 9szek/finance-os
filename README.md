# Finance OS

Boss Sek Finance OS is a finance and accounting web app prototype for daily store reporting, owner review, and admin user/PIN management.

## Current MVP

- Staff daily finance intake by store
- User ID and PIN login
- Admin user and PIN management
- Owner dashboard for all stores
- Revenue, expense, cash, capital, and inventory review
- Local data persistence for prototype usage
- Daily finance import workflow from 2026-01-01 onward
