# SOP - Record Completed Work

## Trigger

Use this when any task is finished.

## Steps

1. Open Notion > Done Log.
2. Create one new item.
3. Fill:
   - Done Item
   - Business
   - Date
   - What Was Done
   - Tools Used
   - Result
   - What Worked
   - What Did Not Work
   - Reuse Potential
   - Convert To
   - Final Files
4. If the work can repeat more than twice, create an SOP.
5. If it is a useful AI command, save it to Prompt Library.
6. If it is a reusable format, save it to Reusable Templates.

## Rule

No finished task should disappear from memory.
