# Inbox Rules

Use this folder only for files that have not been classified yet.

Clear this folder every week.

Decision rule:

- Business-specific file -> `01_BUSINESS_HQ/[Business]/...`
- Shared brand file -> `02_SHARED_ASSETS`
- Finance file -> `03_FINANCE_REPORTS`
- SOP file -> `04_SOP_LIBRARY`
- Prompt file -> `05_PROMPT_LIBRARY`
- Research source -> `06_RESEARCH_ARCHIVE`
- Software/product file -> `07_PRODUCT_SOFTWARE`
- Legal/admin file -> `08_ADMIN_LEGAL_DOCS`
- Final output -> `09_FINAL_OUTPUTS` or business `08_Final Deliverables`

Do not let this folder become permanent storage.
