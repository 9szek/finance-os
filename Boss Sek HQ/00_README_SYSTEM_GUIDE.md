# Boss Sek HQ - Second Brain v1

## Core Rule

Notion = knowledge, decisions, SOP, prompts, done log.
Google Drive / this folder = original files and final deliverables.
NotebookLM = reading room for many documents or long sources.

## Where To Look

- Old work summary: Notion > Done Log
- Business overview: Notion > Business HQ
- Reusable process: Notion > SOP Library
- Reusable AI command: Notion > Prompt Library
- Original/final files: Boss Sek HQ folders
- Large document research: NotebookLM, then save summary back to Notion

## File Naming

Use:

```text
YYYY-MM-DD_Business_Project_FileType_Version
```

Examples:

```text
2026-05-07_SK-Suki_May-Promo_Menu-v1.pdf
2026-05-07_9szek_Press-Kit_Final.pdf
2026-05-07_Evara_SOP-Booking-v1.docx
```

## Finish Every Task

After a task is done:

1. Save final files to the right folder.
2. Add one item to Notion > Done Log.
3. If reusable, convert it to SOP, Prompt, Template, or Workflow.
4. If it came from research, save the summary to Research Notes.

## Weekly Maintenance

- Clear `00_INBOX`.
- Move final files into business folders.
- Review Current Priorities.
- Convert repeated work into SOP or Prompt.
- Delete or archive what is no longer useful.
